# Linux-System-Monitor
System Monitor for Linux (similar to htop)

How to Run:

Install "ncurses" package
```
sudo apt-get install libncurses5-dev libncursesw5-dev
```

Compile and Run
```
g++ -std="c++17" main.cpp -lncurses && ./a.out
```
